template<typename TIdx, typename TConcurrentExec, template< typename TFnObjReturn > class TPromise, typename TYield, typename TMutex, typename TCondVar>
class alpaka::core::detail::ConcurrentExecPool< TIdx, TConcurrentExec, TPromise, TYield, TMutex, TCondVar, false >

ConcurrentExecPool using a condition variable to wait for new work.

Template Parameters
TConcurrentExecThe type of concurrent executor (for example std::thread).
TPromiseThe promise type returned by the task.
TYieldUnused. The type is required to have a static method "void yield()" to yield the current thread if there is no work.
TMutexThe mutex type used for locking threads.
TCondVarThe condition variable type used to make the threads wait if there is no work.
