13  /** Most common Philox engine variant, stateless, outputs a 4-vector of floats
14  *
15  * This is a variant of the Philox engine generator which outputs a vector containing 4 floats. The counter
16  * size is \f$4 \times 32 = 128\f$ bits. Since the engine returns the whole generated vector, it is up to the
17  * user to extract individual floats as they need. The benefit is smaller state size since the state does not
18  * contain the intermediate results. The total size of the state is 192 bits = 24 bytes.
19  *
20  * Ref.: J. K. Salmon, M. A. Moraes, R. O. Dror and D. E. Shaw, "Parallel random numbers: As easy as 1, 2, 3,"
21  * SC '11: Proceedings of 2011 International Conference for High Performance Computing, Networking, Storage and
22  * Analysis, 2011, pp. 1-12, doi: 10.1145/2063384.2063405.
23  *
24  * @tparam TAcc Accelerator type as defined in alpaka/acc
25  */
