60 // PGI and NV HPC SDK compiler detection
61 // As of Boost 1.74, Boost.Predef's compiler detection is a bit weird. Recent PGI compilers will be identified as
62 // BOOST_COMP_PGI_EMULATED. Boost.Predef has lackluster front-end support and mistakes the EDG front-end
63 // for an actual compiler.
64 // TODO: Whenever you look at this code please check whether https://github.com/boostorg/predef/issues/28 and
