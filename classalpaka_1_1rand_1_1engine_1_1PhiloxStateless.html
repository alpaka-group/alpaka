Detailed Description

template<typename TBackend, typename TParams>
class alpaka::rand::engine::PhiloxStateless< TBackend, TParams >

Class basic Philox family counter-based PRNG

Checks the validity of passed-in parameters and calls the TBackend methods to perform N rounds of the Philox shuffle.
