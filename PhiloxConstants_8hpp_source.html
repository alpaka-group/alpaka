20  /** Constants used in the Philox algorithm
21  *
22  * The numbers are taken from the reference Philox implementation:
23  *
24  * J. K. Salmon, M. A. Moraes, R. O. Dror and D. E. Shaw, "Parallel random numbers: As easy as 1, 2, 3,"
25  * SC '11: Proceedings of 2011 International Conference for High Performance Computing, Networking,
26  * Storage and Analysis, 2011, pp. 1-12, doi: 10.1145/2063384.2063405.
27  *
28  * @tparam TParams basic Philox algorithm parameters
29  *
30  * static const data members are transformed into functions, because GCC
31  * assumes types with static data members to be not mappable and makes not
32  * exception for constexpr ones. This is a valid interpretation of the
33  * OpenMP <= 4.5 standard. In OpenMP >= 5.0 types with any kind of static
34  * data member are mappable.
35  */
