21  * Storage and Analysis, 2011, pp. 1-12, doi: 10.1145/2063384.2063405.
22  *
23  * @tparam TParams basic Philox algorithm parameters
24  *
25  * static const data members are transformed into functions, because GCC
26  * assumes types with static data members to be not mappable and makes not
