Detailed Description

template<typename TDim, typename TIdx>
class alpaka::AccCpuOmp2Blocks< TDim, TIdx >

The CPU OpenMP 2.0 block accelerator.

This accelerator allows parallel kernel execution on a CPU device. It uses OpenMP 2.0 to implement the grid block parallelism. The block idx is restricted to 1x1x1.
