Detailed Description

template<typename TBackend, typename TParams, typename TImpl>
class alpaka::rand::engine::PhiloxBaseCommon< TBackend, TParams, TImpl >

Common class for Philox family engines

Checks the validity of passed-in parameters and calls the TBackend methods to perform N rounds of the Philox shuffle.

Template Parameters
TBackenddevice-dependent backend, specifies the array types
